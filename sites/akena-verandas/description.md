# Akena Vérandas

**URL** : https://www.akena.com
**URL form** : https://www.akena.com/demande-de-devis

## Activité

Fabricant français de vérandas, extensions de maison, pergolas, pergolas bioclimatiques, carports, pool-houses et abris de piscine sur-mesure. Usine en Vendée (Dompierre-sur-Yon, 85), réseau d'agences en France et présence Belgique / Suisse / Espagne / Italie. Labels « La French Fab » et « Fabriqué en France ». Plus de 40 ans d'expérience. Configurateur 3D en ligne + RDV à domicile avec un technicien-conseil.

## Style UI / palette

- **Couleur dominante de marque** : noir pur `#000000` + jaune soleil saturé `#F0C828` (mesurés sur le logo).
- **Logo** : silhouette noire stylisée en forme de **petite maison à pignon arrondi**, avec à l'intérieur un grand **cercle jaune coupé en deux par une diagonale** (évoque un soleil traversant une fenêtre, ou un disque solaire). Identité graphique très moderne, contrastée, presque pop, qui se distingue radicalement des codes "véranda" traditionnels (qui sont d'habitude blanc/gris/bleu pâle).
- **Univers général** : photographies haute définition de vérandas et pergolas habitées, beaucoup de blanc pour mettre en valeur la lumière (cohérent avec le métier — laisser entrer le soleil), couleurs vives sur les CTA, ton commercial pro et énergique, slogan implicite "la lumière en plus". Site très architecturé, énorme menu de navigation par produit / usage / style / prix / surface. Vouvoiement professionnel chaleureux.

## Formulaire (résumé)

~10 champs : nom, prénom, email, téléphone, adresse, code postal, ville, pays (dropdown 9 pays UE), **produit qui intéresse** (dropdown 12 options : Véranda / Extension de maison / Pergola-Préau / Carport / Carport solaire / Poolhouse / Abri piscine / Véranda piscine / Sas d'entrée / Stores véranda / Stores pergola / Panneaux solaires), message libre, opt-in marketing, RGPD. Le vrai sur-mesure se fait ensuite via le configurateur 3D séparé et la visite à domicile d'un technicien-conseil sous 48 h.

## Pertinence pour notre projet

Cas du **fabricant industriel sur-mesure** avec gamme de produits large. La qualification clé tourne autour du **type de produit** (12 options) plus la **localisation pour assignation à une agence régionale**. Le contraste graphique noir / jaune est un défi pour un agent IA en marque blanche : il ne faut pas mal interpréter ce code comme "sombre / pessimiste" — c'est au contraire un choix très lumineux et solaire. Le robot d'agent devra refléter cette joie solaire (par exemple traits jaunes sur fond noir, ou logo soleil sur l'avatar).
